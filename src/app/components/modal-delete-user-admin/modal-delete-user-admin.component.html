Restricción de Seguridad
Acción restringida

Has intentado eliminar una cuenta con privilegios de Administrador.

Por políticas de seguridad del sistema, no es posible eliminar directamente cuentas administrativas.

Para continuar, por favor:
  1. 1 Cambia el rol a Usuario estándar
  2. 2 Luego intenta eliminar la cuenta nuevamente

Esta restricción protege la integridad del sistema y previene eliminaciones accidentales.
